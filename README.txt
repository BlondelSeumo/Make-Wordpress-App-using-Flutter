# Cirilla - Multipurpose Flutter Wordpress App.

**Step 1:**

Download the source code in your profile => downloads

**Step 2:**

Go to app/cirilla folder and get dependencies: 

```
flutter pub get 
```

**Step 3:**

Run app:

```
flutter run
```

### Docs & Support
